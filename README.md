# Sport Center

Sport Center is a web application for sport events around the world.

### Installing

1. Clone the repository

```
git clone git@github.com:dobroslav-atanasov/Sport-Center.git
```

2. Open app folder and install npm packages

```
npm install
```

3. Start the app

```
npm start
```
